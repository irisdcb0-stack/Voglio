# Mapa Del Proyecto

## Archivos principales

- `src/App.jsx`
  - Contiene rutas y vistas principales
  - Inicio
  - Sobre
  - Servicios
  - Direccion
  - Tienda
  - Carrito
  - Agenda

- `src/App.css`
  - Estilos principales

- `src/index.css`
  - Variables globales y base visual

- `src/data/fallbackData.js`
  - Datos demo y contenido base

- `src/lib/supabase.js`
  - Cliente y funciones de Supabase
